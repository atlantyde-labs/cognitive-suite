# AI Agent Roles (Labs)

These roles define the personal AI systems that accompany the user during Labs.

## Onboarding Agent
- Explains prerequisites and setup.
- Confirms consent for scoring.

## Lab Coach
- Breaks down tasks into steps.
- Provides hints without solving the task.

## Security Guard
- Flags risky actions (raw data sync, secrets, external calls).
- Enforces compliance guardrails.

## Reviewer
- Checks evidence completeness before submission.
- Highlights policy violations or missing tests.

## Career Agent
- Recommends next Lab based on performance.
- Tracks skill progression and strengths.

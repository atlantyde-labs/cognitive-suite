# Operations Dashboard

The main Dashboard is the command center of the Cognitive Suite, providing a
real-time view of system health and cognitive results.

## üìä Panel components

### 1. System status (Enclave Health)
Shows local connectivity, deployed service versions, and uptime.
* **KPIs**: Node availability, vector database latency, and event broker
  status.

### 2. Analysis metrics
Dynamic visualization of workload:
* **Processed documents**: Total count and weekly evolution.
* **Efficiency**: Average response times by document type.
* **Categorization**: Semantic distribution of detected insights (Risks,
  Opportunities, Entities).

### 3. Recent GitOps activity
A live feed of synchronizations with linked repositories:
* Merge conflict alerts.
* Automatic deployment status.
* Commit history generated by AI.

## Quick Installation

<div class="quickstart-section" markdown="1">

<h3 class="quickstart-title">
Get started in 3 steps
</h3>

Deploy Atlantyqa in your local environment in minutes:

```bash
# 1. Initialize the environment
python cogctl.py init

# 2. Ingest your first document
python cogctl.py ingest <file>

# 3. Run cognitive analysis
python cogctl.py analyze
```

<p style="margin-top: 1.5rem;">
<strong>First time?</strong> Check our <a href="portal/quickstart/">quickstart guide</a> for a step-by-step setup.
</p>

## System Architecture

Atlantyqa is designed with a modular architecture that guarantees scalability and maintainability:

- **Ingestor**: Multimodal content processing and extraction
- **Pipeline**: Semantic analysis and cognitive classification
- **Vectorstore**: Vector storage and embedding search
- **GitOps Agent**: Automatic synchronization with repositories
- **Frontend**: Web interface for visualization and management

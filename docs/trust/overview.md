# Trust & Governance

ATLANTYQA demuestra gobernanza en cada despliegue: controles claros, evidencia continua y modelos reproducibles. Este documento agrupa los assets clave para instituciones que exigen confianza operativa.

## Contenido del Trust Pack

1. One-pagers sectoriales (UE, municipal, corporativo).  
2. Security & Compliance overview con SBOM/SCA, logs, hardening y data residency.  
3. Modelos de despliegue micro-CPD + edge.  
4. Casos de uso con KPIs de auditoría y operaciones.

## Compromiso de implantacion y calidad

- Cualquier implantacion de oferta publica se define con un minimo de **180 dias**.
- Se reserva un buffer operativo de **30 dias** para incidencias inesperadas.
- El cierre de cada fase exige gates de calidad: UAT, seguridad, compliance y evidencias trazables.

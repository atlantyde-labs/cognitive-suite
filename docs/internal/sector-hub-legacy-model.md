# Sector Hub Legacy Model (Internal)

Este documento conserva el marco historico de mensajeria:

- Audiencia
- Narrativa
- Oferta inicial
- Valor

para no perder su valor cognitivo, semantico, ontologico y taxonomico durante la evolucion del hub publico.

## Fuente canonicamente versionada

- Archivo: `knowledge/contracts/sector-hub-legacy-model.yml`
- Alcance: uso interno (enablement, RFP, respuestas a stakeholders, gobernanza de contenido)
- Confidencialidad: interna

## Relacion con el hub publico actual

El hub publico (`docs/sectors/sector-hub.md` y `docs/sectors/sector-hub.en.md`) usa un modelo orientado a conversion:

- Problema a resolver / The challenge
- Resultado esperado / What you gain
- Despliegue inicial / How we deploy
- Impacto esperado / Expected impact

El contrato legacy mantiene la correspondencia y bindings de taxonomia por sector para preservar trazabilidad historica y reutilizacion interna por rol.

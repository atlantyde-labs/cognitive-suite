# Workshop notes compartidas

Archivo colaborativo para que Kabehz y L0oky documenten decisiones cognitivas, emociones y aprendizajes cada vez que ejecuten el workflow de segmentación. El script `scripts/workshop-learning-digest.py` ya deja bloques iniciales con los campos “Observaciones cognitivas” y “Estado emocional y energía”; completa esos campos tras cada run mientras comparten pantalla o al repasar el video de la sesión.

## Registro mínimo

- **Objetivo del run**: (por ejemplo, validar targets secret/internal, ajustar paths, revisar digest en pair-programming).  
- **Decisiones cognitivas clave**: qué se aprendió sobre la mentalidad del equipo, qué heurísticas funcionaron o fallaron.  
- **Estado emocional y energía**: cómo se sintió cada contribuidor tras el push, qué facilitó concentración o causó fricción.
- **Acciones de refuerzo**: ¿qué microlearning digest/quizzes del notebook deben revisarse después?

## Log de sesiones

Actualiza esta sección manualmente si deseas más contexto gráfico (enlace a grabaciones, capturas, etc.). Puedes usar el siguiente formato para cada ejecución:

```
### YYYY-MM-DD · run/dry-run · target
- Notas rápidas
- Emociones
- Aprendizajes para el próximo sprint
```

El archivo también sirve como punto de partida para alimentar los prompts del módulo de bienestar que se sincroniza con Open Notebook.

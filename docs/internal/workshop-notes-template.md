# Registro de sesión compartida

Usa este bloque cada vez que terminen una ejecución en la videollamada. Copia y pega en `docs/internal/workshop-notes.md` para mantener el historial sincronizado antes de que el workshop continúe.

```
### YYYY-MM-DD · [dry-run|run] · Objetivo: <secret/internal/public>
- Digest generado: `outputs/workshop-digests/<timestamp>/digest.md`
- Observaciones cognitivas:
  1. ¿Qué metáforas o modelos mentales explicaron mejor la segmentación?
  2. ¿Qué heurísticas se probaron y cuál fue su impacto?
 3. ¿Qué pasos del script necesitaron aclaración?
- Estado emocional y energía:
  1. ¿Cómo se sintió el equipo (curiosidad, alerta, frustración, motivación)?
  2. ¿Qué alentó la concentración o generó fricción?
  3. ¿Se percibió agotamiento o energía renovada?
- Acciones de refuerzo:
  1. Revisar quiz/resumen de Open Notebook en `*notebook-response.txt`.
  2. Ajustar visibilidad/paths en `scripts/repo-visibility-plan.json`.
  3. Anotar seguimiento en el issue del GitHub Lab Playbook.
```

Documentar este bloque evita pérdida de contexto y facilita alimentar el módulo cognitivo/EMR que se alimentará con Open Notebook.

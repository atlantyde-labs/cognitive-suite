## Instalación Rápida

<div class="quickstart-section" markdown="1">

<h3 class="quickstart-title">
Comienza en 3 pasos
</h3>

Despliega Atlantyqa en tu entorno local en minutos:

```bash
# 1. Inicializa el entorno
python cogctl.py init

# 2. Ingesta tu primer documento
python cogctl.py ingest <archivo>

# 3. Ejecuta el análisis cognitivo
python cogctl.py analyze
```

<p style="margin-top: 1.5rem;">
<strong>¿Primera vez?</strong> Consulta nuestra <a href="portal/quickstart/">guía de inicio rápido</a> para una configuración paso a paso.
</p>

## Arquitectura del Sistema

Atlantyqa está diseñado con una arquitectura modular que garantiza escalabilidad y mantenibilidad:

- **Ingestor**: Procesamiento y extracción de contenido multimodal
- **Pipeline**: Análisis semántico y clasificación cognitiva
- **Vectorstore**: Almacenamiento y búsqueda vectorial de embeddings
- **GitOps Agent**: Sincronización automática con repositorios
- **Frontend**: Interfaz web para visualización y gestión
